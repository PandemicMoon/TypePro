## Keyboard Shortcuts
| Shortcut | Effect                |
| -------- | --------------------- |
| CTRL + C | Copy                  |
| CTRL + X | Cut                   |
| CTRL + V | Paste                 |
| CTRL + A | Select All            |
| CTRL + Z | Undo                  |
| CRTL + Y | Redo                  |
| CRTL + N | Close File (New File) |
| CRTL + O | Open File             |
| CRTL + S | Save File             |
| CRTL + Q | Save File As          |
| CRTL + P | Print File            |
| F5       | Insert Date And Time  |
